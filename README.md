# 💼 Projeto: Perfil de Usuário

Interface simples para exibição, edição e exclusão de perfis de usuário, com frontend em HTML/CSS/JS e backend em PHP.

---

## 🛠️ Tecnologias utilizadas

- HTML5 + CSS3 (Tailwind)
- JavaScript vanilla
- PHP (WAMP, XAMPP, etc.)
